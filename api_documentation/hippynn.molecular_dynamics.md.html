class Variable(name: str, data: dict[str, Tensor], model_input_map: dict[str, str] = {}, updater: VariableUpdater | None = None, device: device | None = None, dtype: dtype | None = None)[source]

Bases: object

Tracks the state of a quantity (eg. position, cell, species, volume) on each particle or each system in an MD simulation. Can also hold additional data associated to that quantity (such as its velocity, acceleration, etc…)

__init__(name: str, data: dict[str, Tensor], model_input_map: dict[str, str] = {}, updater: VariableUpdater | None = None, device: device | None = None, dtype: dtype | None = None) → None[source]
Parameters:

  • name – name for variable

  • data – dictionary of tracked data in the form value_name: value

  • model_input_map – dictionary of correspondences between data tracked by Variable and inputs to the HIP-NN model in the form hipnn-db_name: variable-data-key, defaults to dict()

  • updater – object which will update the data of the Variable over the course of the MD simulation, defaults to None

  • device – device on which to keep data, defaults to None

  • dtype – dtype for float type data, defaults to None

class VariableUpdater[source]

Bases: object

Parent class for algorithms that make updates to the data of a Variable during each step of an MD simulation.

Subclasses should redefine __init__, pre_step, post_step, and required_variable_data as needed. The inputs to pre_step and post_step should not be changed.

__init__(force_db_name: str, units_force: float = 1.0, units_acc: float = 1.0)[source]
Parameters:

  • force_db_name – key which will correspond to the force on the corresponding Variable in the HIPNN model output dictionary

  • units_force – amount of eV equal to one in the units used for force output of HIPNN model (eg. if force output in kcal, units_force = ase.units.kcal = 2.6114e22 since 2.6114e22 kcal = 1 eV), by default ase.units.eV = 1, defaults to ase.units.eV

  • units_acc – amount of Ang/fs^2 equal to one in the units used for acceleration in the corresponding Variable, by default units.Ang/(1.0 ** 2) = 1, defaults to ase.units.Ang/(1.0**2)
